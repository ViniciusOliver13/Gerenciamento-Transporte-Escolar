# Gerenciamento-Transporte-Escolar

Sistema de gerenciamento para transporte escolar, facilitando o controle de rotas, alunos e motoristas.

## Como contribuir

1. Clone o repositório:
   ```bash
   git clone https://github.com/ViniciusOliver13/Gerenciamento-Transporte-Escolar.git

2. Crie seu ambiente virtual:
   ```bash
   python -m venv venv
   source venv/bin/activate  # No Windows use `venv\Scripts\activate`
   ```
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
